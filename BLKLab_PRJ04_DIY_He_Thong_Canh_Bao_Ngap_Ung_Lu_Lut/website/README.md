# Website giới thiệu PRJ04

Mở file `index.html` hoặc `components.html` trong trình duyệt để xem website tĩnh cục bộ. Ví dụ mở bằng command line:

```
# từ thư mục website
python3 -m http.server 8000
# sau đó mở http://localhost:8000
```

Hoặc mở trực tiếp file `index.html` hoặc `components.html` bằng trình duyệt (một số tính năng clipboard yêu cầu HTTPS/localhost để hoạt động hoàn toàn).
