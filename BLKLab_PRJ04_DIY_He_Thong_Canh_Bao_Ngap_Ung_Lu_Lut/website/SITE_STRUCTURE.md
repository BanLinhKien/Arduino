# Cấu trúc trang website giới thiệu dự án PRJ04

- `index.html` — Trang chính, chứa các phần: Hero, Giới thiệu, Tính năng, Nguyên lý, Đoạn mã, Danh sách tệp.
- `styles.css` — Kiểu dáng (theme tối, responsive layout).
- `script.js` — Tương tác nhỏ: cuộn mượt và sao chép đoạn mã.
- `README.md` — Hướng dẫn mở trang cục bộ.

Mô tả từng phần trên `index.html`:

- Hero: tiêu đề ngắn gọn, kêu gọi hành động.
- Giới thiệu: mục tiêu và phần cứng chính của dự án.
- Tính năng: các điểm nổi bật rút từ `FEATURE_IMPROVEMENTS.md`.
- Nguyên lý & đấu nối: tóm tắt chân kết nối từ file .ino.
- Đoạn mã: trích đoạn logic chính để người xem nhanh hiểu flow.
- Tệp dự án: danh sách file quan trọng để tham khảo trong repo.
